# ARIMAX + EGARCH + LLM Sentiment Forecasting

This project tests whether LLM-extracted sentiment improves return and volatility forecasting in financial time series using:

- ARIMAX (conditional mean)
- EGARCH (conditional variance)
- Strict time-safe text alignment
- Rolling out-of-sample windows
- Diebold-Mariano forecast tests
- Backtests w/ transaction costs
